5 HTML tags I have learnt and their description

About me page.
The form tag
Form is a paired tag. Any information we want to collect using this form will be added inside the opening and closing form tags.
The Image tag
The img tag needs two pieces of information one which image to link to and two a short description of the image.
The Ordered List tag
Based on the name ordered lists are used when the order of items in the list is important.
The unordered List tag
An unordered list is used when the items inside the list do not have a specific order or the order is not important.
The Description List tag
This is a special type of list where we have a term and description of the term.
